## What It Does

SkillBarter is a full-stack web platform where users can offer skills they have and request skills they want to learn вҖ” creating a peer-to-peer skill exchange network.

No money changes hands. You teach what you know. You learn what you don't.

---

## The Problem It Solves

Online learning platforms charge money. Tutors charge money. But there are millions of people who have valuable skills and are willing to trade them for other skills they need.

SkillBarter connects those people directly.

## Features

- **User registration + login** вҖ” JWT-based authentication
- **Skill profile** вҖ” list skills you offer and skills you want
- **Browse skills** вҖ” search and filter other users by skill
- **Match requests** вҖ” send and receive barter requests
- **Accept / decline** вҖ” manage incoming skill exchange requests
- **Session tracking** вҖ” keep track of active and completed exchanges

## Getting Started

### Backend
```bash
cd skillbarter-backend
npm install

# Add your MongoDB URI to .env
echo "MONGO_URI=your_mongodb_connection_string" > .env
echo "JWT_SECRET=your_secret_key" >> .env

npm start
# Runs on http://localhost:5000
```

### Frontend
```bash
cd SkillBarter-frontend
npm install
npm start
# Runs on http://localhost:3000
```
