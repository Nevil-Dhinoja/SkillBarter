# SkillBarter

SkillBarter is a platform where users can trade skills, connect with matches, chat, and take quizzes to showcase expertise.

## Features
- Authentication (Login/Register)
- Dashboard with performance charts
- Skill management and tagging
- Matching system
- Chat interface
- Profile management
- Quizzes

## Tech Stack
- React + Vite
- React Router
- TailwindCSS (or CSS modules depending on setup)
- ESLint

## Getting Started
1. Install dependencies:
```bash
npm install
```
2. Run the development server:
```bash
npm run dev
```
3. Build for production:
```bash
npm run build
```
4. Preview production build:
```bash
npm run preview
```

## Project Structure
- `src/pages` contains pages like `Dashboard.jsx`, `Login.jsx`, `Register.jsx`, `Matches.jsx`, `Chat.jsx`, `Profile.jsx`, `Quizzes.jsx`, `Skills.jsx`.
- `src/components` contains reusable components like `SidebarNav.jsx`, `Topbar.jsx`, `TagInput.jsx`, `PerformanceChart.jsx`.
